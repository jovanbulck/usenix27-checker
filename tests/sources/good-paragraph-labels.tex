% =============================================================================
% good-paragraph-labels -- run-in bold labels must not be mistaken for
% squeezed section headings.
%
% A run-in lead-in ("\textbf{Ethical testing.} body...", or \paragraph{}) is a
% bold, body-size label whose paragraph text continues on the same line. Labels
% that begin with a word the heading detector knows ("Ethical...", "Open
% Science...", "Availability...") would otherwise match SECTION_RE, and a tight
% \textbf run-in sits only ~4 pt below the previous line -- exactly the gap a
% squeezed section heading would show. Several appear back to back so a
% regression trips the 2-heading threshold. The checker must skip them because
% each has body text on the same baseline to its right.
%
% Two flavours are covered: tight \textbf run-ins (small gap above, the
% false-positive case) and \paragraph{} labels (larger built-in gap above).
%
% Expected checker outcome: PASS with zero findings.
% =============================================================================
