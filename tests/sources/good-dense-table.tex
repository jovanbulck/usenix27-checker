% with tightly packed, body-size rows (here \arraystretch 0.5) produces many
% small baseline gaps; counting those would wrongly flag the whole page as
% having compressed leading. The checker must sample leading from prose lines
% only (wide lines that fill the column), not from narrow table cells.
%
% Expected checker outcome: PASS with zero findings.
% =============================================================================
